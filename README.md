# FTB Server Installer Scripts
This zip contains the FTB Server Installer Scripts for the modpack FTB OceanBlock 2 version 1.18.4.

If on Windows, run the Batch script `install.bat` to install the server.
If on Linux or MacOS run the shell script `install.sh` to install the server.

Once this script has been run, you will have a server installer file, run that to install the server.

If you have any issues specifically with this script, please report them on the [FTB Server Installer Scripts GitHub](https://go.ftb.team/server-scripts-support)

If you have any issues, please report them to the FTB Team on our [Discord](https://go.ftb.team/discord)
