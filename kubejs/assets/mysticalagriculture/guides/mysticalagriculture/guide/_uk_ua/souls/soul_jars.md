# Банки для душ

Банки для душ використовуються разом з [Соулієвим Кинжалом](./soulium_dagger.md) для збирання душ з мобів. Заповнені банки можна використовувати для створення [Ресурсових культур](../basics/resource_crops.md) відповідної істоти. 

Банки для душ можна наповнити за допомогою дропу з мобів використовуючи [Витягувач Душ](../machines/soul_extractor.md).
