# Удобрююча есенція

Удобрююча есенція **зазвичай** випадає з усіх [Ресурсових культур](./resource_crops.md), **крім Інферієвих**. Працює ідентично кістковій муці, також можна застосовувати на [Ресурсові культури](./resource_crops.md)! 

Удобрюючу есенцію можна використовувати для створення [Містичного Добрива](../advances/mystical_fertilizer.md).
