# Пробуджений супремій

Пробуджений супремій - це матеріал високого рівня, який використовується для створення поліпшених версій Супремієвих предметів. 

Пробуджений супремій **обычно** створюється шляхом наповнення [Пил Знань ](./cognizant_dust.md) з Супремієвимм Блоком на [Вівтарі Пробудження](./awakening_altar.md).
