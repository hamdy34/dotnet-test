# Вівтар пробудження

Вівтар пробудження - це структура, яка **зазвичай** використовується для створення [Пробудженого Супремія](./awakened_supremium.md). Структура складається з одного вівтаря пробудження, 4-ох п'ядесталів пробудження і 4-ох [посудин для есенції](./essence_vessel.md). 

Як тільки Ви розкладете рецепт, потрібно буде просто активувати вівтар сигналом Редстоуна.
