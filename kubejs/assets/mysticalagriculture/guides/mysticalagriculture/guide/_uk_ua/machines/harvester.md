# Комбайн

Комбайни використовуються для автоматичного збору посівів, що повністю виросли. Вони працюють на твердому паливі та мають внутрішній буфер для зберігання енергії. 

Комбайн автоматично пересаджує зібрані культури. 

Комбайн помістить усі ресурси у свій внутрішній інвентар. Якщо місця немає, предмети будуть залишені у місці збору.

## Створіння

Комбайн витрачатиме трохи енергії щоразу, коли перевірятиме чи виросли посіви. Збір посівів, що повністю виросли, споживає більше енергії. 

Цю машину можна поліпшити за допомогою [Поліпшення для машин](./machine_upgrades.md).
