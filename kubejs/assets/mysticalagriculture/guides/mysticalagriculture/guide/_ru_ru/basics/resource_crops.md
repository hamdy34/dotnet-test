# Ресурсовые культуры

Ресурсовые культуры - это сердце и душа того, что делает Mystical Agriculture столь великим. Однако есть некоторые особенности ресурсовых культур, которые следует отметить. 


- Ресурсовые культуры нельзя выращивать с использованием костной муки. 
- С ресурсовых культур нельзя получить второе семечко, если они не посажены на [пашню из эссенции](./essence_farmland.md).

## Создание

Ресурсовые культуры **обычно** создаются с помощью [Алтаря Наполнения](./infusion_altar.md).
