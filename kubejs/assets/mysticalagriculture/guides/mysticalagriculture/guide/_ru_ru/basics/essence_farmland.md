# Пашня из эссенции

Пашня из эссенции увеличивает урожайность культур из Mystical Agriculture несколькими способами. Они перечислены на следующей странице. 

Вы можете создать пашню из эссенции, щёлкнув ПКМ по пашне в мире с эссенцией в руке или создать её.


- С посевов, посаженных на любой уровень пашни из эссенции, шанс выпадения дополнительного семечка составляет 10%%. 
- У посевов, уровень которых совпадает с пашней, на которую они посажены, шанс выпадения доп. семечка составляет 20%%. 
- Инфериевые семена будут приносить больше эссенции, если их посадить на пашню из эссенции более высокого уровня.
