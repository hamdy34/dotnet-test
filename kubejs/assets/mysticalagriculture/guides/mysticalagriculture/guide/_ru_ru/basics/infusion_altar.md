# Алтарь наполнения

Алтарь наполнения - это ремёсленная структура, которая **обычно** используется для создания семян из Mystical Agriculture. Структура состоит из 1 Алтаря Наполнения и 8 Пьедесталов Наполнения. 

После того как вы выложите рецепт, вам нужно будет просто активировать алтарь с помощью сигнала редстоуна.
