# Удобряющая эссенция

Удобряющая эссенция **обычно** выпадает со всех [Ресурсовых Культур](./resource_crops.md), **кроме Инфериевых**. Работает идентично костной муке, также можно применять на [Ресурсовых Культурах](./resource_crops.md)! 

Удобряющую эссенцию можно использовать для создания [Мистического Удобрения](../advances/mystical_fertilizer.md).
