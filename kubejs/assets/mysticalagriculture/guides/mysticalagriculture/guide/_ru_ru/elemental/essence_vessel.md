# Сосуд для эссенции

Сосуды для эссенции используются в структуре [Алтаря Пробуждения](./awakening_altar.md) для хранения эссенций природы, необходимых для создания предмета. Каждый сосуд может содержать до 40 единиц одной эссенции.
