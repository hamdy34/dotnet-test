# Пробуждённый супремий

Пробуждённый супремий - это материал высокого уровня, используемый для создания улучшенных версий Супремиевых предметов. 

Пробуждённый супремий **обычно** создаётся путём наполнения [Пыли Знаний](./cognizant_dust.md) с Супремиевым Блоком на [Алтаре Пробуждения](./awakening_altar.md).
