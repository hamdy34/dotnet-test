# Пыль знаний

Пыль знаний **обычно** можно получить убив Эндер-дракона или визера с помощью [Оружия из эссенции](../tinkering/essence_tools.md), зачарованного на Мистическое Просветление. 

Более высокие уровни Мистического Просветления увеличивают количество пыли, выпадающей при убийстве.
