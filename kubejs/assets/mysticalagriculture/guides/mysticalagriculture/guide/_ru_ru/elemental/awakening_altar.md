# Алтарь пробуждения

Алтарь пробуждения - это структура, которая **обычно** используется для создания [Пробуждённого Супремия](./awakened_supremium.md). Структура состоит из 1-го алтаря пробуждения, 4-ёх пьедесталов пробуждения и 4-ёх [сосудов для эссенции](./essence_vessel.md). 

Как только Вы разложите рецепт, нужно будет просто активировать алтарь сигналом Редстоуна.
