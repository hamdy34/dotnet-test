# Банки для душ

Банки для душ используются вместе с [Соулиевым Кинжалом](./soulium_dagger.md) для сбора душ с мобов. Заполненные банки можно использовать для создания [Ресурсовых Культур](../basics/resource_crops.md) соответствующего существа. 

Банки для душ можно наполнить с помощью дропа с мобов используя [Извлекатель Душ](../machines/soul_extractor.md).
