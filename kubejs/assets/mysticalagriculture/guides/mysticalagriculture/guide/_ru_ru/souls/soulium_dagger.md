# Соулиевый кинжал

Соулиевый кинжал используется для сбора душ с мобов. Эти души собираются с помощью [банок для душ](./soul_jars.md). Души собираются при убийстве мобов с помощью Соулиевого Кинжала 

Соулиевый кинжал можно улучшить с помощью [Алтаря Наполнения](../basics/infusion_altar.md), повысив скорость сбора душ.
