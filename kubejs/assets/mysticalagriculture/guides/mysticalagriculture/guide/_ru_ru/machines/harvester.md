# Комбайн

Комбайны используются для автоматического сбора полностью выросших посевов. Они работают на твёрдом топливе и имеют внутренний буфер для хранения энергии. 

Комбайн автоматически пересаживает собранные культуры. 

Комбайн поместит все ресурсы в свой внутренний инвентарь. Если места нет, предметы будут оставлены в месте сбора.

## Создание

Комбайн будет расходовать немного энергии каждый раз, когда будет проверять выросли ли посевы. Сбор полностью выросших посевов потребляет больше энергии. 

Эту машину можно улучшить с помощью [Улучшений для машин](./machine_upgrades.md).
