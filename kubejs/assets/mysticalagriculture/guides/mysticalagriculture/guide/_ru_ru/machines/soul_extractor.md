# Извлекатель душ

Извлекатели душ используются для преобразования дропа мобов в души мобов. Они работают на твёрдом топливе и имеют внутренний буфер для хранения энергии. 

Извлекатель душ позволяет использовать дополнительный дроп с мобов, который у вас может быть, но обычно не так эффективен, в сравнении с [Соулиевым Кинжалом](../souls/soulium_dagger.md).
