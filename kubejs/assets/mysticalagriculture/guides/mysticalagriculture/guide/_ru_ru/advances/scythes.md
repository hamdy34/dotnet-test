# Косы

Косы используются для сбора посевов на больших площадях, с помощью клика ПКМ. Любые полностью выросшие посевы в зоне действия будут собраны без выкорчевывания. 

Косы также являются эффективным оружием с увеличенной областью атаки. 

Косы более высокого уровня имеют большую площадь действия.
