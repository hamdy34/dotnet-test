# Расширения

Расширения - это улучшения [Инструментов из эссенции](./essence_tools.md) и [Брони из эссенции](./essence_armor.md). Вы можете добавить расширения на своё снаряжение с помощью [Сборочного стола](./tinkering_table.md). 

Каждому расширению присвоен уровень, обозначающий минимальный уровень снаряжения, необходимого для его использования. 

## Создание

В разделе «Расширения» данного руководства вы можете узнать, что делает каждое расширение.
