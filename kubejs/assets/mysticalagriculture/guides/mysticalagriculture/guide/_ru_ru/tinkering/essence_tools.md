# Инструменты из эссенции

Улучшение алмазных инструментов с помощью эссенции повышает их прочность и скорость добычи. Это также позволит надевать на них [Расширения](./augments.md). 

Инструменты из эссенции можно починить на наковальне с помощью соответствующего слитка эссенции. Каждый из этих инструментов имеет 1 слот для расширений.
