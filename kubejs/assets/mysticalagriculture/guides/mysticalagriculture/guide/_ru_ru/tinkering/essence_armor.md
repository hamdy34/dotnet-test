# Броня из эссенции

Улучшение алмазной брони с помощью эссенции повышает её прочность и скорость добычи. Это также позволит надевать на неё [Расширения](./augments.md). 

Броню из эссенции можно починить на наковальне с помощью соответствующего слитка эссенции. Каждая часть брони имеет 1 слот для расширений.
