# Область трамбовки (Расширение)

Область трамбовки (Расширение) - это расширение для лопаты, увеличивающее область до 9х9, в которой лопата утрамбовывает блоки (Например: дёрн в тропу). Расширение работает только на лопате и только с зажатым SHIFT.
