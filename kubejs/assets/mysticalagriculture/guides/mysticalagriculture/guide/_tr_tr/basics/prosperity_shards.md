# Prosperity Parçaları

Prosperity Parçaları, Mystical Agriculture'daki iki temel malzemeden diğeridir. Temel malzemelerin büyülü varyasyonlarını oluşturmak için kullanılırlar. Prosperity Parçaları **genellikle** yerden kazarak elde edebilirsiniz.
