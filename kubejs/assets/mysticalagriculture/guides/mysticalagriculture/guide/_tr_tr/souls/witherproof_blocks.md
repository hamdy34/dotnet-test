# Wither'a Dayanıklı Bloklar

Wither'a Dayanıklı Bloklar, Wither tarafından kırılamaz. Patronları durdurmayı sevenler için çok kullanışlıdır. 

Ayrıca Ender Ejderhası'nın hasarına karşı bağışıklıkları vardır, ancak yine de içlerinden uçabilir.
