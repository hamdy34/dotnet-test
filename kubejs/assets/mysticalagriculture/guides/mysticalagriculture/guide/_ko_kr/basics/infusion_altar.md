# 주입 제단

주입 제단은 **보통** 신비 농업 씨앗을 만드는 데 사용되는 조합 구조입니다. 이 구조는 주입 제단 1개와 주입 받침대 8개로 구성됩니다. 

일단 조합법을 설치했으면 레드스톤 신호로 제단을 작동시키면 됩니다.
