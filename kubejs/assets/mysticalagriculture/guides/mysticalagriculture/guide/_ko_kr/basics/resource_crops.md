# 자원 작물

자원 작물은 신비농업을 위대하게 만드는 것의 핵심입니다. 하지만, 주목해야 할 자원 작물에는 몇 가지 특별한 속성이 있습니다. 


- 뼛가루를 사용하여 자원 작물을 재배할 수 없습니다. 
- 자원 작물은 [에센스 경작지](./essence_farmland.md)에 심지 않는 한 두 번째 씨앗을 떨어뜨리지 않습니다.

## 조합

자원 작물은 **일반적으로** [주입 제단](./infusion_altar.md)을 사용하여 생성됩니다.
