# 부적

[에센스 도구](./essence_tools.md)와 [에센스 갑옷](./essence_armor.md)의 업그레이드입니다. [땜장이 작업대](./tinkering_table.md)를 사용하여 장비에 부적을 추가합니다. 

각 부적에는 장비에 필요한 최소 장비 조각 티어를 지정하는 티어가 할당됩니다.
