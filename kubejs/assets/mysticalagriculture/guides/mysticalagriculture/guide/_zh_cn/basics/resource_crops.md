# 资源作物

资源作物是让神秘农业如此成功的核心和灵魂. 但是, 还有一些应该注意的资源作物的特殊属性. 


- 资源作物不能通过使用骨粉来加速生长. 
- 资源作物不会掉落第二个种子除非被种在[精华耕地](./essence_farmland.md)上.

## 合成

资源作物**通常**使用[注魔祭坛](./infusion_altar.md)来合成.
