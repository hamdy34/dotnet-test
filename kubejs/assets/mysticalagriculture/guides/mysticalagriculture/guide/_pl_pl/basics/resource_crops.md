# Uprawy materiałów

Uprawy materiałów są sercem i duszą Mystical Agriculture. Są jednak pewne wyjątkowe cechy upraw materiałów, o których powinniśmy wspomnieć. 


- Wzrost upraw materiałów nie może być przyspieszony mączką kostną. 
- Po zebraniu plonów z uprawy materiału nie wypada dodakowe nasiono, chyba że jest posadzona na [polu uprawnym z esencji](./essence_farmland.md).

## Konstruowanie

Uprawy materiałów są **zazwyczaj** wytwarzane z pomocą [ołtarza infuzji](./infusion_altar.md).
