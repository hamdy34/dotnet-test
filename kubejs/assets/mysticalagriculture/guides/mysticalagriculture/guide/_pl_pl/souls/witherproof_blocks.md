# Witheroodporne bloki

Witheroodporne bloki są niezniszczalne dla Withera. Bardzo przydatne dla tych, którzy nie chcą zbytnio się wysilać przy walce z bossami. Są też niezniszczalne dla Smoka Endu, jednak wciąż może on przez nie przelecieć.
