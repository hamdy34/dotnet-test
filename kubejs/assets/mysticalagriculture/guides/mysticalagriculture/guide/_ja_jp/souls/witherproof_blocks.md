# 防ウィザーブロック

防ウィザーブロックは、ウィザーが破壊できないブロックです。ボスに地形を穴だらけにされたくない人にとっては非常に便利です。

エンダードラゴンもこのブロックを破壊できませんが、すり抜けて飛行するのであまり意味はありません。
