# 資源の作物

資源の作物は、Mystical Agricultureの核心です。しかし、資源の作物には特筆すべきいくつかの特性があります。


- 資源の作物は、骨粉を使用して成長させることができません。
- 資源の作物は、[エッセンスの耕地](./essence_farmland.md)に植えない限り複数の種をドロップしません。

## クラフト

資源の作物は**通常**、[注入の祭壇](./infusion_altar.md)を使用して作成されます。
