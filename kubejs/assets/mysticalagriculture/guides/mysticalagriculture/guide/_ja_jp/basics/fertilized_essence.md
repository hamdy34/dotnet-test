# 肥料のエッセンス

肥料のエッセンスは**通常**、**インフェリウムを除く**すべての[資源の作物](./resource_crops.md)からドロップします。骨粉と同じように機能しますが、[資源の作物](./resource_crops.md)にも使用することができます！

肥料のエッセンスは、[神秘の肥料](../advances/mystical_fertilizer.md)を効率的にクラフトするためにも使用できます。
