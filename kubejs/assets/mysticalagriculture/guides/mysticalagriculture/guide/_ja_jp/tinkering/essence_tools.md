# エッセンスの道具

ダイヤモンドの道具をエッセンスで強化すると、耐久値と採掘速度が上昇します。また、[オーグメント](./augments.md)を装着できるようになります。

エッセンスの道具は、対応するエッセンスのインゴットを使用して金床で修理することができます。また、これらの道具にはそれぞれ1つのオーグメントスロットがあります。
