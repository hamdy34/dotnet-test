# 肥料精華

肥料精華**通常**從所有[資源作物](./resource_crops.md)中掉落, 除了**下級精華**. 它的工作原理與骨粉相同, 但也同樣作用於[資源作物](./resource_crops.md)! 

肥料精華也可以用來更高效地合成[魔法肥料](../ advances/mystical_fertilizer.md).
