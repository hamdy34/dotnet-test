# 資源作物

資源作物是讓神秘農業如此成功的核心和靈魂. 但是, 還有一些應該注意的資源作物的特殊屬性. 


- 資源作物不能通過使用骨粉來加速生長. 
- 資源作物不會掉落第二個種子除非被種在[精華耕地](./essence_farmland.md)上.

## 合成

資源作物**通常**使用[注魔祭壇](./infusion_altar.md)來合成.
